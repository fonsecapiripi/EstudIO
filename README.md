# EstudIO
Una aplicacion con el objetivo de ayudarte a gestionar tu proceso de estudio. Creas una materia y empezas a tomar notas.
La idea es incluir recordatorios, tareas, para que no te olvides de nada.
Básicamente, virtualizamos tu escritorio y te facilitamos el seguimiento de tareas.
Usa:
  1) Tauri (RUST) + Vanilla TS
  2) Archivos .md, para permitir texto e imagenes
